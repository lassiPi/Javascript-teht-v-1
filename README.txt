Javascript perusteet, tehtävä 1


	"palindromiluesyöte.js" ajettaessa komentoriviltä kysyy käyttäjältä syötettä ja
	kertoo, onko se palindromi.
	
	"palindromipelkkäfunktio.js" sisältää palindromi(sana)-funktion, joka palauttaa
	kutsuttaessa totuusarvon riippuen siitä, onko "sana" palindromi vai ei.
	
	"palindromitesti.html" on yksinkertainen sivu, jolla voi testata onko tekstiriville
	kirjoitettu sana palindromi.
	
	"palindromistyle.css" on edellistä varten luotu .css-tiedosto.
